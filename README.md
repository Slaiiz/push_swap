# push_swap

Syntax: ./push_swap [-e] [-v] [-c] [-g] n1 [ ... [nn]]

FLAGS:

* -e  : Errors. Print accurate description of errors
* -v  : (partial) Verbose. Print state of both stacks for each action taken.
